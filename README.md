[English](#english) | [中文](#中文)

## English

Office document processing toolkit for Claude Code - PDF, Word, PowerPoint, Excel manipulation and document co-authoring workflows.

### Overview

This skill collection provides comprehensive document processing capabilities, enabling Claude Code to create, edit, and analyze Office documents programmatically.

### Skills Included

| Skill | Description |
|-------|-------------|
| `pdf` | PDF manipulation toolkit - extract, create, merge, split, forms |
| `docx` | Word document processing - create, edit, format |
| `pptx` | PowerPoint presentations - create, edit, analyze |
| `xlsx` | Excel spreadsheets - formulas, formatting, data analysis |
| `doc-coauthoring` | Document co-authoring workflow guide |

### Installation

```bash
cd ~/.claude/skills
git clone https://github.com/LeoLin990405/claude-office-skills.git
```

### Usage

```
/pdf              # Work with PDF files
/xlsx             # Create or edit spreadsheets
/pptx             # Build presentations
/doc-coauthoring  # Collaborative document workflow
```

### Features

- **PDF**: Extract text/tables, create new PDFs, merge/split, fill forms
- **Word**: Create documents with formatting, tables, images
- **PowerPoint**: Build slides with layouts, charts, animations
- **Excel**: Formulas, pivot tables, charts, data validation

## 中文

Office 文档处理工具集 - PDF、Word、PowerPoint、Excel 操作及文档协作工作流。

### 概述

此技能集提供全面的文档处理能力，使 Claude Code 能够以编程方式创建、编辑和分析 Office 文档。

### 包含的技能

| 技能 | 描述 |
|------|------|
| `pdf` | PDF 操作工具包 - 提取、创建、合并、拆分、表单 |
| `docx` | Word 文档处理 - 创建、编辑、格式化 |
| `pptx` | PowerPoint 演示文稿 - 创建、编辑、分析 |
| `xlsx` | Excel 电子表格 - 公式、格式、数据分析 |
| `doc-coauthoring` | 文档协作工作流指南 |

### 安装

```bash
cd ~/.claude/skills
git clone https://github.com/LeoLin990405/claude-office-skills.git
```

### 使用方法

```
/pdf              # 处理 PDF 文件
/xlsx             # 创建或编辑电子表格
/pptx             # 构建演示文稿
/doc-coauthoring  # 协作文档工作流
```

### 功能特性

- **PDF**: 提取文本/表格、创建新 PDF、合并/拆分、填写表单
- **Word**: 创建带格式、表格、图片的文档
- **PowerPoint**: 构建带布局、图表、动画的幻灯片
- **Excel**: 公式、数据透视表、图表、数据验证
