# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [2.0.0] - 2026-03-16

### Added
- Cross-format workflows: data-to-presentation, document-review-cycle, pdf-form-processing
- Document structure templates: report-structure, proposal-structure, meeting-notes
- Intent-based routing layer for auto-detecting document type
- CONTRIBUTING.md with contribution guidelines
- GitHub issue templates (bug report, feature request)
- GitHub pull request template
- CHANGELOG.md

### Changed
- Restructured repository following github-repo-design best practices
- Upgraded README.md with badges, feature table, project structure tree, and bilingual content

## [1.0.0] - 2025-01-01

### Added
- Initial release with 5 document processing skills
- PDF skill: extract, create, merge, split, fill forms
- Word (docx) skill: create, edit, format with OOXML schema support
- Excel (xlsx) skill: formulas, charts, pivot tables, recalculation
- PowerPoint (pptx) skill: slides, layouts, charts, HTML-to-PPTX conversion
- Document co-authoring skill: collaborative editing and review
- OOXML schemas (ISO-IEC 29500 and Microsoft extensions)
- Python and JavaScript helper scripts
